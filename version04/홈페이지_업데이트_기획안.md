## 3. 브랜드 · 언어 정책 (확정)

### 3.1 브랜드명 — HypePotions 유지
- 메인 브랜드는 그대로 **HypePotions** 사용 (nav, hero, footer 타이틀 등)
- PDF 소개서의 운영주체인 **Sentience**는 footer에 "HypePotions by Sentience" 형태로 병기하여 신뢰도(법인/운영사) 정보 제공
- 연락처는 기존 Calendly 링크 유지, 필요 시 contact@sentience.rocks 추가 고려(footer 하단 소문구 정도로)

### 3.2 언어 정책 — 한/영 토글
- 사이트는 **영문을 기본 언어**로 유지하되, 우측 상단 nav에 **언어 토글 버튼(KO/EN)** 추가
- 모든 텍스트 콘텐츠(Hero, 섹션 타이틀, 카드 설명, FAQ 등)는 한/영 두 버전을 데이터로 보유하고 JS로 전환
- 구현 방식 제안: `data-ko` / `data-en` 속성 또는 `i18n` JSON 객체 + `app.js`에서 토글 시 텍스트 교체 (페이지 새로고침 없이 즉시 전환)
- 숫자/통계(예: `▲434/mo`, `+536%`)는 언어 무관하게 공통 표기, 레이블만 번역
- 기본 진입 언어는 영문(EN) 유지, 사용자가 토글 시 한글(KO)로 전환 후 `localStorage`에 선택 언어 기억 권장

### 4.1 Hero (유지)
- 기존 카피("Save Your 100 Hours of Marketing") 및 서브카피, CTA 버튼 구성 그대로 유지
- **통계 바는 넣지 않음** — 첫 화면은 메시지와 행동 유도(CTA)에 집중하고, 정량적 증거는 바로 다음 섹션(Proof/Key Stats)에서 보여주는 흐름으로 설계
- 한/영 토글에 따라 hero 카피도 즉시 전환

### 4.2 Proof / Key Stats (신규 — 통계 통합 섹션)
Hero 다음 섹션에서 회사 규모 통계와 성과 지표를 한 번에 보여줍니다 (PDF 1페이지 + 4페이지 데이터 통합).

**회사 규모/경력 (PDF 1p)**
- `20+` 파트너 타이틀(Titles Partnered)
- `15개국` 글로벌 마켓 커버(Global Markets)
- `▲434/mo` 평균 위시리스트 속도 증가분(Avg. Wishlist Growth)
- `5년+` 스팀 마케팅 경력(Years of Experience)

**성과 지표 (PDF 4p, 6p)**
- `+536%` 평균 팔로워 성장률
- `27.28%` 평균 게시물 인게이지먼트
- `60+` PR 매체 네트워크(11개국·7개 언어)
- `▲677/mo` 최고 위시리스트 증가 실적(Best Case)

- 8개 통계를 2열(또는 4열) 그리드로 배치, 작은 주석: "*Metrics vary by channel / title."

### 4.3 What We Do (4대 핵심 서비스로 개편)
현재 7개 서비스 카드를 PDF의 **4대 핵심 서비스** 체계로 재구조화(각 카드에 세부 항목을 접거나 나열):

1. **Social Media Management** — X/Reddit/TikTok/YouTube 계정 운영 & 커뮤니티 빌딩 (성과: +536% 팔로워)
2. **Press Release & PR** — 유료 피처링(Famitsu, 4Gamer 등) + 직접 배포 (평균 20회 노출/타이틀)
3. **Online Event Support** — Steam Next Fest, Steam Sale, 3rd party 이벤트 기획·실행
4. **Paid Social Ads** — Reddit/TikTok/YouTube 타겟팅 광고 + Steam Analytics 연동 성과 측정

> 기존 7개 세부 서비스(타겟 리서치, 스팀 페이지 최적화 등)는 4대 카테고리의 하위 항목으로 흡수.

### 4.4 Results / 실적 사례 (신규)

## 5. 디자인 가이드

- **색상 시스템 유지**: 기존 CSS 변수(`--color-highlight-purple #6600FF`, `--color-highlight-green #00FFA3`) 그대로 사용 — PDF와 동일하므로 추가 작업 불필요
- **데이터 카드 스타일**: PDF의 통계 카드(큰 숫자 + 작은 레이블) 패턴을 신규 섹션에 적용. 큰 숫자는 그린/보라 강조색 활용
- **숫자 카운트업 애니메이션**: 통계 섹션 진입 시 0 → 목표값 카운트업 효과 권장 (`app.js`에 추가)
- **반응형**: 신규 통계/네트워크 섹션은 모바일에서 1열로 자연스럽게 무너지도록 grid 설계
- **라이트 vs 다크**: 현재 라이트 테마 유지가 기본. 단, Proof/Results 섹션을 다크 배경 블록으로 처리하면 PDF 분위기와 대비 효과를 동시에 줄 수 있음(선택)
- **언어 토글 UI**: nav 우측(Book a Call 버튼 옆 또는 그 앞)에 작은 `KO / EN` 텍스트 토글 또는 globe 아이콘 버튼 배치. 모바일 메뉴에도 동일하게 노출
